Hexo+Github Pages建站指北

发表于 2018-04-15

之前一直在OneNote上面写笔记。结果每次笔记同步的时候，我的Microsoft账号老是连不上。想着去年买的个人域名一直没有用，干脆就搭一个博客吧。

搭建博客的教程网上一抓一大把，这里选用比较常见的方案：Hexo+GitHub Pages.

  • Hexo是什么？
    Hexo中文网站
    拿官网的说法，hexo是一个快速、简洁且高效的博客框架。（嗯，使用过后确实相当便捷~ o(￣▽￣)o）

  • Github Pages是什么？
    Github Pages官方网站
    简单来说，Github Pages提github供给用户用来展示个人或者项目主页的静态网页系统。非常棒的服务，为我们穷学生省下了租服务器的钱。

有了Hexo和Github Pages，搭建博客相当的便捷。具体方法前面的网站写的很详细，这里简单说一下吧~
